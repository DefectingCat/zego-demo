## Zego express demo

Zego express demo with React 18.

## Set token

Create or edit `.env` file:

```
REACT_APP_userId1='user1'
REACT_APP_roomId1='1'
REACT_APP_userName1='user1'
REACT_APP_token1='token'

REACT_APP_userId2='user2'
REACT_APP_roomId2='1'
REACT_APP_userName2='user2'
REACT_APP_token2='token'
```
